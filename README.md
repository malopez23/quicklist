# Quicklist

Aplicação web simples para gerenciar uma lista de compras da semana.

## Deploy

Acesse a versão publicada na Vercel:

https://quicklist-tawny.vercel.app

## Tecnologias

- HTML
- CSS
- JavaScript

## Funcionalidades

- Adicionar item à lista
- Remover item da lista
- Marcar e desmarcar item como concluído
- Exibir alerta ao remover item
